ALERTES PRIORITAIRES

Cette section regroupe toutes les alertes critiques necessitant une action immediate du superviseur :

- Equipements en retard d'inspection
- Certifications expirees
- Items manquants apres un deploiement
- Anomalies detectees sur le terrain

Les alertes sont triees par priorite. Traitez-les dans l'ordre affiche.

URGENCES TEMPS REEL

Flux en direct des urgences declenchees par les equipes sur le terrain via le bouton d'urgence de l'application.

Les urgences sont synchronisees en temps reel via Firebase Firestore et apparaissent instantanement ici.

Le badge rouge indique le nombre d'urgences recentes non resolues.

CERTIFICATIONS

Suivi des 10 certifications obligatoires pour chaque membre : RCR, PDSB, SIMDUT, Nacelle, Chariot, Espace clos, Travail en hauteur, Sauvetage, Electricite, Premiers secours.

Vert = A jour | Jaune = Expire dans 30 jours | Rouge = Expire

Le badge indique le nombre de certifications expirees ou bientot expirees.

TRACABILITE AUDIT

Journal chronologique de toutes les transactions (PICK-ON, PICK-OFF, modifications) pour assurer la tracabilite complete de l'inventaire.

Chaque entree enregistre : l'operateur, la date/heure, les items concernes et le type d'operation.

Exportable en CSV pour les rapports CNESST et les audits internes.

INVENTAIRE CRITIQUE

Vue d'ensemble des equipements par statut avec le graphique donut. Identifie rapidement :

- Repartition disponible / sorti / inspection / maintenance
- Items avec date d'expiration proche
- Equipements necessitant un remplacement

Essentiel pour la planification des achats et la conformite CNESST.
