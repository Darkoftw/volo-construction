Questions frequentes

C'est quoi exactement que je recois?

Un fichier ZIP avec le code source complet, structure et commente. Chaque template inclut un README avec les instructions d'installation et de personnalisation. Vous pouvez deployer en moins de 5 minutes.

Je peux utiliser le code pour mes clients?

Oui, la licence commerciale est incluse. Vous pouvez utiliser les templates pour vos projets personnels et pour vos clients sans limites. La seule restriction: ne pas revendre les templates tels quels.

Ca fonctionne avec quel framework?

Les templates HTML/CSS sont en vanilla — zero framework requis. Le Firebase starter utilise le SDK officiel. Les scripts Python utilisent des librairies standard (requests, firebase-admin). Pas de dependances exotiques.

Les templates sont-ils optimises pour le SEO?

Oui. HTML semantique, meta tags, Open Graph, schema markup, images optimisees, score Lighthouse 95+. Tout est en place pour un bon classement Google des la mise en ligne.

Est-ce que je recois les mises a jour?

Oui. Le Kit 3 inclut les mises a jour gratuites et le Bundle Dev inclut les mises a jour a vie. Vous recevez un email quand une nouvelle version est disponible avec le lien de telechargement.
